# cats

## Required
https://nodejs.org/en/download/

## Project setup
Go to folder with code and type in console
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
